# Task Manager Challenge

Este proyecto es una solución para el reto planteado por la empresa de software Perficient, diseñado para ayudarte a organizar y gestionar tus tareas diarias de manera eficiente. La aplicación de administración de tareas te permite registrar, categorizar, establecer fechas de vencimiento y recibir recordatorios para garantizar que cumplas con tus responsabilidades de manera oportuna.

## Características Principales

### 1. Registro de Tareas
Permite registrar nuevas tareas con detalles específicos, como nombre, descripción, categoría y fecha de vencimiento.

### 2. Categorización y Colores
Puedes asignar categorías a tus tareas y diferenciarlas visualmente mediante colores. Esto facilita la identificación rápida de las tareas en función de su naturaleza.

### 3. Recordatorios
Recibe recordatorios automáticos 3 días antes, 2 días antes y 1 día antes de la fecha de vencimiento de tus tareas, para mantenerte informado y preparado.

### 4. Edición y Eliminación
Tienes la capacidad de editar y actualizar la información de tus tareas en cualquier momento. También puedes eliminar tareas que ya no son relevantes.

### 5. Información Adicional
Ingresa información relevante a cada tarea, como direcciones, nombres de médicos o enlaces importantes para tener toda la información necesaria a tu alcance.

### 6. Consultas Avanzadas
Consulta el estado de tus tareas según su progreso: pendientes, en progreso, canceladas o finalizadas.

### 7. Visualización por Periodo
Explora tus tareas del día, semana o mes para tener una visión clara de tus responsabilidades a corto y largo plazo.

### 8. Filtrado por Categoría o Estado
Filtra tus tareas pendientes según la categoría o el estado, permitiéndote enfocarte en áreas específicas de tu vida o trabajo.

### 9. Vencimientos Próximos
Consulta las tareas pendientes que vencen en los próximos 5 días, asegurándote de estar preparado para cumplir con tus compromisos.

## Uso del Proyecto

1. Clona este repositorio en tu máquina local.
2. Instala las dependencias necesarias utilizando el archivo `requirements.txt`.
3. Ejecuta la aplicación principal y comienza a gestionar tus tareas.
